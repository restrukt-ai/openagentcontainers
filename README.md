# Open Agent Containers Spec

Specification for the Open Agent Containers (OAC) standard.

## Docs

This directory is a [MkDocs](https://www.mkdocs.org) project. To serve locally:

```sh
pip install -r requirements.txt
mkdocs serve
```

The documentation covers the OAC label schema — how to annotate a Docker image so that a compliant
orchestrator can provision its LLM, credentials, workspaces, and event subscriptions automatically.
